# 💼 Personal Portfolio

A responsive personal portfolio website built using HTML, CSS, JavaScript, and Bootstrap. The portfolio showcases my education, technical skills, projects, and contact information through a clean and modern user interface.

## 🌐 Live Demo

🔗 https://srijabala.github.io/Portfolio/

## 📂 GitHub Repository

🔗 https://github.com/srijabala/Portfolio

## 📖 Overview

This portfolio was created to present my academic background, technical skills, and web development projects in a simple and professional format. It features responsive layouts, smooth navigation, and an organized structure to provide visitors with an overview of my learning journey and projects.

---

## ✨ Features

- Responsive navigation menu
- Hero section with personal introduction
- About Me section
- Education timeline
- Technical skills showcase
- Project showcase with project links
- Contact section with social media links
- Contact form
- Mobile-friendly responsive design

---

## 🛠️ Technologies Used

- HTML5

- Bootstrap 5
- Font Awesome

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/srijabala/Portfolio.git
```

Navigate to the project directory:

```bash
cd Portfolio
```

Open `index.html` in your preferred web browser.
